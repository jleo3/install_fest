install_fest
============

To start:
1) Grab the script
Linux -
```wget https://raw.github.com/jleo3/install_fest/master/install_fest.sh```

Mac -
```curl https://raw.github.com/jleo3/install_fest/master/install_fest.sh -O```

2) Make it executable
```chmod +x install_fest.sh```

3) Run
```./install_fest.sh```
